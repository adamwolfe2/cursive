Key Features
============

Schema Markup:
  • BlogPosting (headline, author, publisher, dates, images)
  • BreadcrumbList (auto-generated from navigation)
  • FAQPage (when FAQs present)

AEO Optimization:
  • Question-based headings (H2/H3)
  • Table of contents with jump links
  • FAQ section with schema
  • Featured snippet optimization
  • Direct answers in first 100 words

User Experience:
  • Reading time calculator (225 WPM)
  • Sticky sidebar TOC (desktop)
  • Social sharing (Twitter/LinkedIn/Facebook/Email)
  • Related posts grid
  • Multiple CTA placements
  • Print-friendly styles

Responsive Design:
  • Mobile-first approach
  • Breakpoints: sm, md, lg, xl
  • Touch-friendly targets
  • Responsive images
  • Collapsible TOC on mobile

Component Quick Reference
=========================

BlogPostLayout
  Purpose: Main layout orchestrator
  Props: post (BlogPost), relatedPosts (array)
  Features: Schema markup, responsive layout, all sub-components

Breadcrumbs
  Purpose: Navigation + schema
  Props: items (array of { name, href })
  Schema: BreadcrumbList

TableOfContents
  Purpose: Sticky TOC with active highlighting
  Props: headings (array from extractHeadings)
  Features: Intersection Observer, smooth scroll

AuthorBox
  Purpose: Author info display
  Props: author (name, role, avatar, bio, social)
  Features: Avatar, bio, social links

CTABox
  Purpose: Varied CTAs throughout content
  Props: variant (demo/trial/pricing/newsletter), custom overrides
  Features: 4 pre-configured variants, icon differentiation

FAQSection
  Purpose: FAQ with schema
  Props: faqs (array of Q&A), pageUrl (string)
  Schema: FAQPage
  Features: Accordion UI, keyboard accessible

RelatedPosts
  Purpose: Related content grid
  Props: posts (array of post summaries)
  Features: 3-column responsive grid, hover effects

SocialShare
  Purpose: Share buttons
  Props: url (string), title (string)
  Features: Twitter, LinkedIn, Facebook, Email, Copy link

Utility Functions
=================

calculateReadingTime(content: string): number
  Returns reading time in minutes (225 WPM average)

formatDate(date: string): string
  Converts ISO 8601 to "February 4, 2026"

extractHeadings(content: string): Heading[]
  Extracts H2/H3 headings for TOC generation

generateShareUrls(url: string, title: string): object
  Returns social share URLs

createSlug(title: string): string
  Converts title to URL-friendly slug

Content Management Options
===========================

1. Static TypeScript/JavaScript (Implemented)
   • Simple, type-safe
   • Good for small blogs
   • See: blog-content-loader.ts

2. Markdown Files (Documented)
   • gray-matter + remark/rehype
   • Version controlled
   • Developer-friendly

3. Headless CMS (Documented)
   • Contentful, Sanity, Strapi
   • Non-technical editing
   • Real-time updates

4. Database (Documented)
   • Supabase, PostgreSQL
   • Full control
   • Dynamic content
