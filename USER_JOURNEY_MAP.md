## Key Metrics to Track

### Stage 1: Landing Page
- **Bounce rate** (goal: <60%)
- **Time on page** (goal: >30s)
- **CTA click rate** (goal: >5%)

### Stage 2-4: Signup & Onboarding
- **Completion rate** (goal: >70%)
- **Time to complete** (goal: <5 min)
- **Drop-off points**

### Stage 5: First Dashboard Visit
- **Path selection rate** (goal: >40%)
- **Time to first action** (goal: <10 min)

### Stage 6: Active User
- **Engagement rate** (goal: >60% weekly active)
- **Leads created** (goal: >5 per week)

### Stage 7: Upgrade
- **Services page visit rate** (goal: >20%)
- **Conversion rate** (goal: >5%)

## Common Pitfalls to Avoid

1. **Too many options** at any stage
2. **Unclear next steps** on dashboard
3. **Upselling too early** (before value)
4. **Abstract language** in messaging
5. **Long onboarding** (>5 min)
6. **No clear path** for new users

---

**Remember:** Every stage should pass the 10-second test. If users are confused, simplify.
